# Summerhill Liquor

A modern e-commerce website for Summerhill Liquor, built with Laravel and Bootstrap. The project includes:

- Product listing, categories, and wishlist functionality  
- User management system  
- Responsive and attractive dashboard with stats and quick actions  
- Interactive UI with icons, hover effects, and progress bars  

## Technologies Used

- PHP (Laravel Framework)  
- MySQL Database  
- Bootstrap 5  
- HTML, CSS, JavaScript
